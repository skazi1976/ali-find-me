🔁 החזרות בשיין (SHEIN) — המדריך המלא

איך מחזירים, האם צריך להדפיס תווית, ומה עושים אם ההחזר תקוע.

בקצרה: מחזירים דרך האפליקציה (ההזמנות שלי → Return Item), בוחרים פריטים וסיבה, ומקבלים תווית או קוד החזרה. לא תמיד צריך להדפיס — בחלק מהשיטות מספיק קוד QR בנייד. ההחזר הכספי מגיע אחרי שהפריט מתקבל בחזרה.

האם חובה להדפיס את תווית ההחזרה?

תלוי בשיטת ההחזרה:

  • תווית להדפסה (Return Label) — מדפיסים ומדביקים על החבילה.
  • קוד QR / קוד החזרה — מראים בנייד בנקודת המסירה, בלי הדפסה.

בודקים מה נדרש בעמוד ההחזרה באפליקציה אחרי שמתחילים את התהליך.

איך מחזירים — שלב אחר שלב

1. אפליקציית שיין → ההזמנות שלי → בוחרים את ההזמנה.
2. לוחצים Return Item (החזרת פריט) ובוחרים את הפריטים.
3. בוחרים סיבה (מידה לא מתאימה / לא תואם וכו') ושיטת החזרה.
4. מקבלים תווית או קוד QR, אורזים ושולחים דרך נקודת המסירה.
5. שומרים את מספר המעקב של ההחזרה ואת אישור המסירה.

כמה זמן לוקח ההחזר הכספי?

אחרי ששיין מקבלת ובודקת את הפריט המוחזר, ההחזר מועבר תוך מספר ימים עד שבועיים. החזר לארנק שיין (Wallet) מהיר יותר מהחזר לכרטיס אשראי.

ההחזרה תקועה / לא קיבלתי החזר — מה עושים?

⏰ אם החבילה המוחזרת התקבלה אבל אין החזר: פונים לשירות הלקוחות של שיין בצ'אט, מצרפים את מספר המעקב של ההחזרה ואת הסטטוס ("התקבל במחסן"), ומבקשים לזרז. שמרו את אישור המסירה — זו ההוכחה שלכם.

שאלות נפוצות

חובה להדפיס תווית החזרה?

לא תמיד — בחלק מהשיטות מספיק קוד QR בנייד. בודקים בעמוד ההחזרה באפליקציה.

איך מחזירים מוצר?

ההזמנות שלי → Return Item → בוחרים פריטים וסיבה → תווית/קוד → שולחים.

כמה זמן לוקח החזר?

מספר ימים עד שבועיים אחרי קבלת הפריט; ארנק שיין מהיר מכרטיס אשראי.

ההחזרה תקועה, מה עושים?

פונים לשיין עם מספר המעקב של ההחזרה ומבקשים לזרז את ההחזר.
